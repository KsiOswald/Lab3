Получить последовательноть из первых цифр чисел, содержащихся в исходной последовательности

На ввод получается первое число, последнее число и шаг между ними, все числа проверяются.

Новая последовательность выводится на экран

## Задача 2



### Текст задачи



Сколько раз в последовательности встречается заданное число? 



### Алгоритм решения



На ввод получается первое число, последнее число и шаг между ними, все числа проверяются.

Затем на основе введённых пользователем данных строится последовательность.

После на ввод получается искомое число, которое также проверяется.

Каждое число сравнивается с искомым числом.

Количество совпадений выводится на экран.
